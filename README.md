# Pearson Quarks

##How to begin?
Clone this repository.<br>
Install node.<br>

Run command:<br>
`npm install`<br>

Run command:<br>
`npm run grunt` - to build and serve the project<br>
or<br>
`npm run grunt -- build` - to build the project<br>

##Assets
CSS is available in:<br>
`dist/assets/styles/pearson-quarks.css`
